# Aider LLM Leaderboards

# Aider LLM 排行榜

## 简介

Aider 擅长与擅长**编写**和**编辑**代码的 LLM 配合工作，并使用基准测试来评估 LLM 在无人干预的情况下遵循指令并成功编辑代码的能力。[Aider 的多语言基准测试](https://aider.chat/2024/12/21/polyglot.html#the-polyglot-benchmark)通过 225 个具有挑战性的 Exercism 编程练习来测试 LLM，涵盖 C++、Go、Java、JavaScript、Python 和 Rust。

## Aider 多语言编程排行榜

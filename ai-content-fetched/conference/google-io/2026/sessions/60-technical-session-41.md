# 使用 Android Studio 的 AI 代理让您的界面自适应

## 📋 摘要

**📅 日期**: 2026-05-21 | **🕐 时间**:  PT | **📂 分类**: Android | **🎯 类型**: 技术演讲 | **📊 等级**: Intermediate

**👤 演讲者**: donturner, jalc

了解 Android Studio 的 AI 代理如何自动让您的 UI 具备自适应能力，从而支持 Android 广泛的设备形态生态系统。我们将演示该代理如何将代码库迁移到使用 Navigation 3——自适应应用的基础。我们还将探讨它如何使用实验性设备能力 API，以及受 Web 启发的自适应布局（如 FlexBox 和 Grid）来为桌面、平板、可折叠设备等优化您的 UI。

## 📝 详细原文

了解 Android Studio 的 AI 代理如何自动使您的 UI 具有自适应能力，以支持 Android 庞大的设备形态生态系统。我们将演示该代理如何将代码库迁移到使用 Navigation 3——自适应应用的基础。我们还将探讨它如何使用实验性设备能力 API，以及受 Web 启发的自适应布局——FlexBox 和 Grid——来优化您的 UI，适用于桌面、平板、可折叠设备等。

**原文（English）**: Discover how Android Studio’s AI agent can automatically make your UI adaptive so that it supports Android’s vast ecosystem of form factors. We’ll demonstrate how the agent can migrate a codebase to use Navigation 3 - the foundation of adaptive apps. We’ll also explore how it can use an experimental device capability API, plus web-inspired adaptive layouts - FlexBox and Grid - to optimize your UI for desktops, tablets, foldables, and more.

### 原文

Discover how Android Studio’s AI agent can automatically make your UI adaptive so that it supports Android’s vast ecosystem of form factors. We’ll demonstrate how the agent can migrate a codebase to use Navigation 3 - the foundation of adaptive apps. We’ll also explore how it can use an experimental device capability API, plus web-inspired adaptive layouts - FlexBox and Grid - to optimize your UI for desktops, tablets, foldables, and more.

## 📝 内容总结

**《使用 Android Studio 的 AI 代理让您的界面自适应》核心要点**

一、核心主题概述

本演讲介绍如何利用Android Studio的AI智能体和自适应Skills让UI适配各种设备形态，涵盖Navigation 3迁移、自适应布局API等。

**二、关键发布/技术要点**
- 要点1：UI能力设计——基于窗口尺寸和指针类型而非设备形态
- 要点2：自适应Skill——指导性指南帮助把UI做成自适应
- 要点3：Compose迁移——View系统迁移到Compose优先
- 要点4：Navigation 3迁移——原生支持多窗格布局
- 要点5：Navigation Suite Scaffold——开箱即用自适应导航
- 要点6：Scene Decorators——自定义应用级布局装饰
- 要点7：Media Query API——CSS媒体查询风格的UI能力访问
- 要点8：Flexbox和Grid——实验性布局API

**三、落地应用**
- 案例1：Paparazzi宠物照片App逐步适配各种屏幕
- 案例2：列表-详情布局替代消息界面切换
- 案例3：Flexbox让筛选条在大屏上换行显示

**四、总结**
让UI自适应不再复杂。通过自适应Skills、Navigation 3、Media Query和Flexbox/Grid等API，配合AI智能体，开发者可轻松构建适配手机、平板、折叠屏、桌面的高质量应用。

# modelcontextprotocol.io

# MCP（模型上下文协议）翻译

MCP（模型 Context Protocol，模型上下文协议）是一个开源标准，用于将 AI 应用程序连接到外部系统。

通过 MCP，像 Claude 或 ChatGPT 这样的 AI 应用程序可以连接到数据源（如本地文件、数据库）、工具（如搜索引擎、计算器）和工作流（如专业提示词）——使它们能够访问关键信息并执行任务。

可以将 MCP 想象成 AI 应用程序的 USB-C 端口。正如 USB-C 提供了连接电子设备的标准化方式一样，MCP 提供了将 AI 应用程序连接到外部系统的标准化方式。

## MCP 能实现什么？

- **智能体**可以访问您的 Google 日历和 Notion，成为更加个性化的 AI 助手。
- **Claude Code**可以使用 Figma 设计生成整个 Web 应用程序。
- **企业聊天机器人**可以连接组织内的多个数据库，使用户能够通过聊天分析数据。
- **AI 模型**可以在 Blender 上创建 3D 设计，并使用 3D 打印机打印出来。

## 为什么 MCP 很重要？

根据您在生态系统中的位置，MCP 可以带来一系列好处：

- **开发者**：MCP 在构建或集成 AI 应用程序或智能体时，可以减少开发时间和复杂性。
- **AI 应用程序或智能体**：MCP 提供了对数据源、工具和应用程序生态系统的访问，这将增强功能并改善最终用户体验。
- **最终用户**：MCP 带来了更强大的 AI 应用程序或智能体，它们可以在必要时访问您的数据并代表您执行操作。

## 广泛的生态系统支持

MCP 是一个开放协议，得到广泛客户端和服务器的支持。Claude 和 ChatGPT 等 AI 助手，以及 Visual Studio Code、Cursor、MCPJam 等开发工具都支持 MCP——这使得一次构建、处处集成变得容易。

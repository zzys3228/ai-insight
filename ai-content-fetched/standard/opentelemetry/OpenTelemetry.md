# 遥测领域的开放标准

OpenTelemetry
是一个面向云原生软件的开源可观测性框架。它提供了一套统一的 API、库、代理和收集器服务，用于从您的应用程序中采集分布式追踪和指标数据。

OpenTelemetry 汇集了 OpenTracing 和 OpenCensus 项目多年积累的经验，并融合了社区中最优秀的理念和实践。

## 厂商无关的插桩

使用 OpenTelemetry API 和 SDK 对代码进行一次插桩，即可将遥测数据导出到任何可观测性后端——无论是 Jaeger、Prometheus、商业供应商还是您自己的解决方案。切换后端无需更改应用程序代码。

## 统一的可观测性信号

通过贯穿整个请求路径的共享上下文，将追踪、指标和日志关联起来。全面了解应用程序在所有组件和服务中的行为表现。

## 任意运行

OpenTelemetry 100% 开源且厂商无关。可在本地部署、混合环境或跨多个云中部署，拥有完全的灵活性且零锁定。将工作负载迁移到任何您需要的地方。

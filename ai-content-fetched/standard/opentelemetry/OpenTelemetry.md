# OpenTelemetry 中文翻译

## 遥测的开放标准

OpenTelemetry 是一个面向云原生软件的开源可观测性框架。它提供一套统一的 API、库、代理和采集器服务，用于从您的应用程序中捕获分布式追踪和指标。

OpenTelemetry 汇集了 OpenTracing 和 OpenCensus 项目多年积累的经验，并结合了社区中最佳的理念与实践。

## 厂商无关的插桩

使用 OpenTelemetry API 和 SDK 为您的代码插桩一次，即可将遥测数据导出到任何可观测性后端——Jaeger、Prometheus、商业供应商或您自己的解决方案。切换后端无需修改应用程序代码。

## 统一的可观测性信号

通过贯穿整个请求路径的共享上下文，将追踪、指标和日志关联起来。全面了解应用程序在所有组件和服务中的行为。

## 随处运行

OpenTelemetry 100% 开源且厂商无关。可在本地、混合环境或多云环境中部署，兼具完全的灵活性和零锁定。可将工作负载移动到任何您需要的地方。

## 可观测性信号

OpenTelemetry 功能特性：

- 自动插桩
- 采集器管道
- 上下文传播
- 多语言支持
- 稳定且已可用于生产
- 开放规范

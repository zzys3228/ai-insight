**MCP 可以实现什么？**

代理（Agent）可以访问您的 Google 日历和 Notion，成为更加个性化的 AI 助手。
Claude Code 可以根据 Figma 设计稿生成完整的 Web 应用。
企业聊天机器人可以连接到组织内的多个数据库，让用户通过聊天来分析数据。
AI 模型可以在 Blender 中创建 3D 设计，并使用 3D 打印机打印出来。

**为什么 MCP 很重要？**

开发者：MCP 降低了构建 AI 应用或代理，或与其进行集成时的开发时间和复杂度。
AI 应用或代理：MCP 提供对数据源、工具和应用程序生态系统的访问，这将增强功能并改善最终用户体验。
最终用户：MCP 带来更强大的 AI 应用或代理，它们可以在必要时访问您的数据并代表您执行操作。

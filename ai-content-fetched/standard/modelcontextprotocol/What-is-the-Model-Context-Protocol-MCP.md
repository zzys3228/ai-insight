# 什么是模型上下文协议（MCP）？

**来源**: modelcontextprotocol.io | **日期**: 2026-06-22

---

## 翻译

**本页内容**

**MCP 能实现什么？**

智能体可以访问您的 Google Calendar 和 Notion，成为更加个性化的 AI 助手。
Claude Code 可以使用 Figma 设计生成完整的 Web 应用。
企业聊天机器人可以连接组织内的多个数据库，帮助用户使用聊天进行数据分析。
AI 模型可以在 Blender 上创建 3D 设计，并使用 3D 打印机打印出来。

**为什么 MCP 很重要？**

**开发者**：MCP 降低了构建或集成 AI 应用或智能体时的开发时间和复杂性。
**AI 应用或智能体**：MCP 提供了对数据源、工具和应用程序生态系统的访问，这将增强功能并改善最终用户体验。
**最终用户**：MCP 带来了更强大的 AI 应用或智能体，它们可以在必要时访问您的数据并代表您执行操作。

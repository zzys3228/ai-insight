# w3c-cg.github.io

# 摘要

另见该文档的[中文版本](https://w3c-cg.github.io/ai-agent-protocol/index_cn.html)。

本文探讨了从语义网（Semantic Web）愿景向新兴的代理网络（Agentic Web）的演进历程，并分析了建立标准化代理网络协议的必要性。尽管二十年前由蒂姆·伯纳斯-李（Tim Berners-Lee）及其合作者提出的语义网概念具有前瞻性，但由于当时人工智能能力的局限性，该愿景未能完全实现。随着大语言模型（LLM）等现代人工智能技术的快速发展，代理（Agent）如今已具备自主执行任务、进行复杂推理和解决多步骤问题的能力，从而催生了代理网络（Agentic Web）。通过系统分析，本文确定了代理网络的四个核心趋势：代理取代传统软件成为互联网基础设施、代理之间的通用互联、基于协议的原生连接模式，以及代理的自主组织和协作能力。同时，研究揭示了当前互联网架构对代理网络发展构成的三大挑战：限制代理决策质量的数据孤岛、阻碍代理交互效率的人机接口，以及缺失标准协议导致的代理协作障碍。针对这些挑战，本文详细阐述了代理网络协议的设计原则和核心需求，并对当前主要的代理网络协议方案（MCP、A2A、ACP、ANP等）进行了系统比较和分析。结论强调，建立标准化的代理网络协议对于打破数据孤岛、实现异构代理协作、构建原生AI数据网络，最终实现开放高效的代理网络至关重要，同时呼吁各相关方积极参与W3C的标准化进程。

# 引言：从语义网未竟的愿景到代理网络的曙光

二十年前，蒂姆·伯纳斯-李及其合作者富有远见地提出了语义网的概念，其核心目标是创建一个以数据为中心、机器可读的万维数据网，使计算机与人类能够更高效地协作。这一概念描绘了一个智能化的未来：日常交易、行政事务和各种生活场景将通过机器对机器的对话由“智能代理”自动完成。为实现这一目标，语义网计划通过XML、RDF和本体（Ontology）等技术为网络上的信息赋予明确的语义定义，使软件代理能够自主穿梭于网页之间，高效代表用户执行复杂任务。

## 框架博客查找方法

### 问题：漏掉LangChain博客、vLLM博客

AI框架官方博客是重要的行业信息来源，发布核心迭代和落地案例。

### 查找方法
```bash
# 搜索框架官方博客
site:blog.langchain.dev
site:vllm.ai blog
site:pytorch.org/blog

# 或直接搜索
{框架名} official blog OR dev blog
```

### 常见框架博客

### 收录规则
| 类型 | 标准 |
|------|------|
| 框架博客 | 有独立博客域名或明显blog板块，稳定更新 |
| 更新频率 | 至少月更 |

## 四重核验

| 维度 | 标准 |
|------|------|
| 内容垂直 | 70%+内容聚焦AI/Agent/具身 |
| 更新时效 | 近3个月有更新 |
| 行业影响力 | 业内广泛订阅/引用 |
| 头部门槛 | 行业主流权威 |

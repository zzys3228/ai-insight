## 检索关键词

```
VC: a16z generative AI report, 红杉中国大模型产业报告
智库: AI Index Report, State of AI robotics forecast
咨询: McKinsey embodied AI, Gartner AI Hype Cycle
政策: EU AI Act guidance, OECD AI risk framework
安全: LLM catastrophic risk, METR autonomous agent
```

## 分类筛选规则

| 分类 | 筛选标准 |
|------|---------|
| VC | 仅收录官方完整行业报告，剔除个人随笔 |
| 智库 | 数据中立，具备可复用底层数据集 |
| 咨询 | 内容聚焦企业商业化落地 |
| 政策 | 仅官方正式法规、标准、征求意见稿 |
| 安全 | 核心聚焦大模型对齐、越狱风险、红队评测 |

## 四重核验

| 维度 | 标准 |
|------|------|
| 机构权威 | 一级VC/知名智库/官方机构 |
| 内容质量 | 有公开报告/白皮书/分析 |
| 更新时效 | 近6个月有更新 |
| 行业认可 | 被广泛引用/采用 |
